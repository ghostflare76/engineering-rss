Meta, Zoomer로 AI 성능 디버깅/최적화 자동화한 방법

GPU 활용률 극대화, 학습 시간 최대 75%↓, 전력 소비 78%↓

수십만 대의 GPU가 바쁘게 돌아가는 Meta의 AI 인프라를 상상해 본 적 있으신가요? 겉으로 보기엔 엄청나게 강력해 보이지만, 이 거대한 시스템 속에서도 미묘한 성능 병목현상 하나가 엄청난 에너지 낭비와 막대한 운영 비용으로 이어지곤 합니다. 최적화되지 않은 GPU 활용률은 단순히 자원 낭비를 넘어, 새로운 기능 출시와 혁신을 지연시키는 주범이 됩니다.

이런 상황에서 수동으로 성능 문제를 진단하고 해결하는 건 거의 불가능에 가깝습니다. 복잡한 AI 워크로드의 특성과 엄청난 규모 때문에, 엔지니어들은 매일매일 숨겨진 비효율성을 찾아내느라 애를 먹을 수밖에 없죠. 단 1%의 효율 개선도 전체 시스템에서는 엄청난 용량 증대와 직결되기에, 이 문제를 해결하는 것이 Meta AI 인프라의 핵심 과제였습니다.

이 글에서는 Meta가 어떻게 이 문제를 해결하기 위해 AI를 위한 포괄적이고 자동화된 디버깅 및 최적화 플랫폼, Zoomer를 개발하고 활용하고 있는지 자세히 살펴봅니다. 특히 다음 내용을 중심으로 알아볼 거예요.

  • Zoomer의 3단계 아키텍처 이해하기 - 인프라, 분석, 시각화 레이어
  • 자동 프로파일링부터 인사이트까지 - Zoomer의 작동 메커니즘
  • 학습, 추론, GenAI 특화 기능 파헤치기 - 고급 최적화 기술

  • Meta AI의 미래를 위한 성능 디버깅 청사진 - 확장성과 지속 가능성

🔎 Meta 규모 AI, 왜 성능 디버깅이 필수일까요?

Meta의 AI 인프라는 광고 랭킹, 콘텐츠 추천, GenAI 기능 등 광범위하고 고도화된 워크로드를 지원하며, 매일 수백조 번의 AI 모델이 실행됩니다. 이 거대한 시스템을 효율적으로 운영하는 것이 가장 중요한데요. GPU 활용률이 낮아지면 모델 학습 반복 주기가 길어져 신제품 출시가 늦어지고, 추론 병목 현상은 사용자의 요청을 제때 처리하지 못하게 만듭니다.

자원 낭비를 줄이고 워크플로우를 가속화하는 것은 더 큰 모델을 효율적으로 학습하고, 더 많은 사용자에게 서비스를 제공하며, 환경 발자국을 줄이는 데 필수적입니다. Meta는 모든 GPU 클러스터에서 단 1%의 활용 개선도 혁신과 성장을 위한 엄청난 용량 증대로 이어진다는 것을 잘 알고 있습니다. 그래서 자동화된 성능 디버깅이 그 어느 때보다 중요해졌죠.

⚙️ Zoomer 아키텍처: 3개의 레이어로 완성된 AI 디버깅 플랫폼

Zoomer는 Meta의 AI 학습 및 추론 워크로드 전반에서 작동하는 자동화된 성능 프로파일링, 디버깅, 분석, 최적화 플랫폼입니다. 에너지 절약, 워크플로우 가속화, 효율성 증대를 위한 심층적인 성능 인사이트를 제공하죠. Zoomer는 세 가지 핵심 레이어로 구성되어 있습니다.

1. 인프라 및 플랫폼 레이어

수십만 GPU를 아우르는 Meta의 방대한 인프라에서 워크로드를 프로파일링하는 데 필요한 엔터프라이즈급 확장성과 안정성을 제공합니다. Manifold (Meta의 블롭 스토리지 플랫폼)를 사용한 분산 스토리지, 대용량 트레이스 파일을 처리하는 내결함성 파이프라인, 수천 대 호스트에서 동시에 작동하는 저지연 데이터 수집 및 자동 프로파일링 트리거를 포함합니다.

2. 분석 및 인사이트 엔진

Kineto 및 NVIDIA DCGM을 통한 GPU 트레이스 분석, StrobeLight 통합을 통한 CPU 프로파일링, dyno 텔레메트리를 통한 호스트 레벨 분석 등 여러 전문 분석기를 통해 심층적인 분석 기능을 제공합니다. 성능 안티패턴을 자동으로 감지하고 실행 가능한 권장 사항을 제시합니다.

3. 시각화 및 사용자 인터페이스 레이어

복잡한 성능 데이터를 직관적이고 실행 가능한 인사이트로 변환합니다. 수천 개의 랭크에 걸친 GPU 활동을 보여주는 대화형 타임라인 시각화, 장기 실행 학습 워크로드를 위한 다중 반복 분석, 장치별 백분위수 분석이 가능한 드릴다운 대시보드, Perfetto와 통합된 트레이스 데이터 시각화 등을 포함합니다. 주요 병목 현상과 최적화 기회를 강조하는 자동화된 인사이트 요약도 제공됩니다.

그림 1. Zoomer의 3가지 핵심 아키텍처 레이어

🚀 Zoomer 작동 원리: 자동 프로파일링부터 맞춤형 인사이트까지

Zoomer는 AI 워크로드 최적화를 위한 정교한 접근 방식을 사용합니다. 프로파일링 트리거부터 분석 파이프라인, 최종 인사이트 도출까지의 과정을 알아볼까요?

  • 프로파일링 트리거 메커니즘: 며칠 또는 몇 주 동안 실행되는 학습 워크로드의 경우, Zoomer는 시작 노이즈를 피하고 안정 상태의 성능을 포착하기 위해 반복 550~555회 정도에 자동으로 프로파일링을 트리거합니다. 추론 워크로드는 즉각적인 디버깅을 위해 온디맨드로 트리거하거나, 지속적인 모니터링을 위해 자동 부하 테스트 시스템과 통합됩니다.
  • 종합적인 데이터 캡처: 각 프로파일링 세션 동안 Zoomer는 여러 데이터 스트림을 동시에 수집합니다. GPU 성능(SM 활용률, 메모리 대역폭, Tensor Core 활용률 등 DCGM), 상세 실행 트레이스(커널 레벨 GPU 작업, 메모리 전송, PyTorch Profiler 및 Kineto), 호스트 레벨 성능(dyno 텔레메트리), 애플리케이션 레벨 주석, 추론 특정 데이터(Crochet profiler, Thrift), 그리고 분산 워크로드를 위한 통신 분석(NCCL)까지 모두 포괄합니다.
  • 분산 분석 파이프라인: 수집된 원시 데이터는 Straggler 감지, Bottleneck 분석(CPU/GPU/메모리/통신 병목), Critical Path 분석, Anti-Pattern 감지, 병렬 처리 분석, 메모리 분석, 부하 불균형 분석 등 다양한 자동 분석을 수행하는 정교한 처리 시스템을 거칩니다.
  • 다중 포맷 출력 생성: 결과는 대화형 타임라인 시각화, 포괄적인 메트릭 대시보드, Perfetto와 통합된 트레이스 뷰어, 핵심 병목 현상 및 권장 사항을 강조하는 자동화된 인사이트 요약, 그리고 제안된 최적화로 작업을 다시 실행할 수 있는 실행 가능한 노트북 등 다양한 인터페이스로 제공됩니다.
  • 특화된 워크로드 지원: GenAI와 같은 대규모 분산 학습을 위해 GPU 효율성 히트맵, N차원 병렬화 시각화 등을 제공하는 목적 구축 플랫폼이 포함되어 있습니다. 추론의 경우, 단일 GPU 모델부터 수천 대 서버에 걸친 대규모 분산 추론까지 모든 것을 분석합니다.

✨ 학습, 추론, GenAI: Zoomer의 고급 최적화 기능

Zoomer는 다양한 AI 워크로드 유형과 규모에 맞춰 설계된 광범위한 고급 기능을 제공합니다. 몇 가지 핵심 기능을 살펴볼까요?

훈련 최적화 파워하우스

  • Straggler 분석: 분산 학습 작업에서 다른 랭크보다 현저히 느린 랭크를 식별하여 동기화 병목 현상으로 인한 전체 작업 지연의 근본 원인을 진단합니다.
  • Critical Path 분석: PyTorch 애플리케이션에서 가장 긴 실행 경로를 식별하여 가장 영향력 있는 최적화 기회에 노력을 집중할 수 있게 합니다.
  • 고급 트레이스 조작: 대용량 트레이스 파일(랭크당 2GB 이상)을 압축, 필터링, 조합, 분할하는 정교한 도구를 제공하여 이전에 처리 불가능했던 대규모 학습 작업 분석을 가능하게 합니다.

추론 효율성 기능

  • 원클릭 QPS 최적화: 병목 현상을 식별하고 자동 부하 테스트를 트리거하여 최적화 시간을 단축합니다. 모델 특성에 따라 2%에서 50%의 QPS 개선을 달성했습니다.
  • 요청 레벨 심층 분석: Crochet 프로파일러와의 통합을 통해 Thrift 요청 레벨 분석을 제공하여 기존 메트릭으로는 놓치기 쉬운 큐 시간 병목 현상 및 서빙 비효율성을 식별합니다.
  • 실시간 메모리 프로파일링: GPU 메모리 할당을 추적하여 메모리 누수, 할당 패턴, 최적화 기회에 대한 실시간 인사이트를 제공합니다.

GenAI 특화 기능

  • LLM Zoomer (대규모): 10만 개 이상의 GPU 워크로드를 지원하는 목적 구축 플랫폼으로, N차원 병렬화 시각화, 수천 개 장치에 걸친 GPU 효율성 히트맵, 텐서, 파이프라인, 데이터, 전문가 병렬화 상호 작용에 대한 전문 분석을 제공합니다.
  • 후처리 워크플로우 지원: SFT, DPO, ARPG 워크플로우를 포함한 GenAI 후처리 작업에 대한 향상된 기능을 제공하며, 생성기와 훈련기 프로파일링을 분리합니다.

범용 지능 기능

  • Holistic Trace Analysis (HTA): 통신 오버헤드, 워크로드 불균형, 커널 비효율성 등 분산 학습 병목 현상을 진단하고 자동 부하 분산 권장 사항을 제공하는 고급 프레임워크입니다.
  • Zoomer Actionable Recommendations Engine (Zoomer AR): 머신러닝 기반 권장 시스템으로 효율성 안티패턴을 자동으로 감지하고, 자동 수정 diff, 최적화 노트북, 제안된 개선 사항으로 작업을 한 번의 클릭으로 다시 시작하는 기능을 생성합니다.
  • 다중 하드웨어 프로파일링: NVIDIA GPU, AMD MI300X, MTIA, 그리고 CPU 전용 워크로드 전반에 걸쳐 하드웨어 플랫폼과 관계없이 일관된 분석 및 최적화 권장 사항을 기본적으로 지원합니다.
